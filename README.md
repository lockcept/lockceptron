## Development

```sh
$ yarn
$ yarn pm2:init
$ yarn start:dev
$ yarn restart:dev # restart
$ yarn stop:dev # stop


$ yarn dev # only if foreground run
```

## Deployment

```sh
$ yarn
$ yarn pm2:init
$ yarn start
$ yarn restart # restart
$ yarn stop # stop
```
